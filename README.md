Checker
=======
Prerequisites
	- install and configure ant 
	- only for javadoc and schemaspy: install http://www.graphviz.org/
	
1. install
	 - cd src
	 - ant -f install.xml
	 
2. configure
	- src/build.properties
	
3. start
	- src
	- ant all (or a specific target)
		
Note: 
	- crap4j works only with java6
